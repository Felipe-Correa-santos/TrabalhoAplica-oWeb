Portfólio

Abaixo estão alguns dos projetos desenvolvidos durante minhas atividades acadêmicas e estudos.

Projetos

  • Banco de dados de locadora de veículos

    Este é um projeto de banco de dados para uma locadora de veículos fictícia, que permite o gerenciamento de informações sobre veículos, clientes, aluguéis e pagamentos pendentes ou pagos, com o objetivo de manter um controle organizado tanto dos veículos quanto dos clientes. O projeto foi desenvolvido para aplicar os conhecimentos adquiridos em aula e melhorar minhas habilidades em modelagem de dados. As linguagens utilizadas foram SQL e MySQL.

    Projeto Banco de Dados de Locadora de Veículos no GitHub

  • Calculadora Simples

    Uma calculadora básica com operações fundamentais, como adição, subtração, multiplicação e divisão. Este projeto foi desenvolvido para melhorar minhas habilidades e desenvolver minha lógica de programação. A linguagem utilizada foi Java. Não cheguei a utilizar POO nem criar uma interface gráfica.

    Projeto Calculadora Simples no GitHub

  • Lista de Contatos

    Este é um projeto de um sistema de cadastro e consulta, que permite ao usuário cadastrar, consultar e remover contatos. O projeto foi totalmente desenvolvido em Python com o objetivo de aplicar os conhecimentos adquiridos em aula.
